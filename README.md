# AI 3D model image generator
This software requires a working OpenAI key that is already paid
What you can do with this:
# 1
Get images to copy to blender or any modeling software
# 2
Have fun using this!
# HOW TO USE THIS:
Write something you want on the input field and then press Enter . Wait at least 2 minutes and then press preview. For your API key do not generate any images you want. There are samples on the outputs folder
